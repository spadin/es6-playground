# ES6-Playground

A simple project to play around with new ES6 features using Traceur.

## Prerequisites

* Node.js
* NPM
* Bower

## Install requirements

    $ npm install
    $ bower install

## Compiling the app

    $ npm run gulp

## Start a server to preview

    $ cd public
    $ python -m SimpleHTTPServer
    $ open http://localhost:8000
